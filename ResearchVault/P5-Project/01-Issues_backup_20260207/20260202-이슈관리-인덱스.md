# 🚨 P5 복합동 이슈 관리

## 📊 이슈 현황 요약

### 🔴 미해결 이슈 (Open / In Progress)
```dataview
TABLE WITHOUT ID
  link(file.link, title) as "제목",
  issue_status as "상태",
  owner as "담당자",
  due_date as "마감일",
  priority as "우선순위"
FROM "P5-Project/01-Issues"
WHERE issue_id AND issue_status != "closed" AND issue_status != "resolved"
SORT priority DESC, due_date ASC
```

### ⚠️ 마감 임박 (7일 이내)
```dataview
TABLE WITHOUT ID
  link(file.link, title) as "제목",
  owner as "담당자",
  due_date as "마감일"
FROM "P5-Project/01-Issues"
WHERE issue_id AND due_date AND due_date <= date(today) + dur(7 days) AND issue_status != "closed"
SORT due_date ASC
```

### 📈 상태별 현황
```dataview
TABLE length(rows) as "건수"
FROM "P5-Project/01-Issues"
WHERE issue_id
GROUP BY issue_status as "상태"
```

### 📁 카테고리별 현황
```dataview
TABLE length(rows) as "건수"
FROM "P5-Project/01-Issues"
WHERE issue_id
GROUP BY category as "카테고리"
```

## ⚙️ 동기화 정보
- **소스**: Google Sheets (AppSheet)
- **스크립트**: `scripts/p5_issue_sync.py`
- **설정**: `_config/p5-sync-config.yaml`

### 동기화 명령
```bash
# Google Sheets에서 동기화
python scripts/p5_issue_sync.py sync

# CSV fallback
python scripts/p5_issue_sync.py sync --csv "path/to/issues.csv"

# 상태 확인
python scripts/p5_issue_sync.py status
```
