# SS-Splice / 기둥이음 공법 P5 적용 검토

## 배경

2026-02-03 삼성 E&A (이광 프로)와 센코어테크 (이승환 대표, 이인호 이사) 간 회의에서 P5 복합동 철골 접합부 관련 두 가지 공법을 논의함.

## 검토 대상 공법

### 1. SS-Splice
- **장점**: 볼트 감소로 인한 설치 간소화
- **단점**: 오차 흡수율 적음

### 2. 기둥이음 공법
- **장점**: 볼트수 감소 + 횡이동 시공오차 흡수 가능
- **단점**: 접합부 플레이트 용접변형

## 검토 요청사항 (센코어테크)

1. P5 적용 가능성 검토
2. Beam 접합부 볼트 감소 방안
3. 기둥이음 공법 P5 적용 검토

## 필요 자료 (삼성 E&A → 센코어테크)

- [ ] Beam 접합부 설계 시 부재력 및 상세 도면
- [ ] 기둥 이음부 설계 시 부재력 및 상세 도면
