# 제텔카스텐 방법론

## 핵심 원칙
1. **원자성(Atomicity)**: 하나의 노트에 하나의 아이디어
2. **자율성(Autonomy)**: 각 노트는 독립적으로 이해 가능해야 함
3. **연결성(Connectivity)**: 모든 노트는 다른 노트와 연결
4. **자기 표현(Own Words)**: 원문을 자신의 말로 재작성

## 노트 유형
| 유형 | 설명 | 수명 |
|------|------|------|
| 임시 노트(Fleeting) | 순간 아이디어 메모 | 1-2일 |
| 문헌 노트(Literature) | 자료 읽기 기록 | 영구 |
| 영구 노트(Permanent) | 정제된 아이디어 | 영구 |
| 프로젝트 노트 | 특정 프로젝트용 | 프로젝트 기간 |

## 작성 프로세스
1. 자료를 읽으며 **문헌 노트** 작성 (핵심만 발췌)
2. 문헌 노트를 바탕으로 **영구 노트** 작성 (자기 말로)
3. 기존 노트와 **링크** 생성
4. 필요시 **MOC** 노트 갱신

## 좋은 영구 노트의 조건
- 제목만 보고 내용을 짐작할 수 있음
- 다른 노트 없이도 단독으로 이해 가능
- 최소 2개 이상의 다른 노트와 연결됨
- 자신의 사고가 반영되어 있음 (단순 복사 X)
