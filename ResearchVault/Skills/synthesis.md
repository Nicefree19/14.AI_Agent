# 종합/분석 스킬

## 목적
여러 노트와 자료를 결합하여 새로운 인사이트 도출

## 종합 기법

### 1. 패턴 인식
- 여러 노트에서 반복되는 주제/개념 찾기
- Obsidian 그래프 뷰로 클러스터 확인
- 태그 검색으로 관련 노트 모으기

### 2. 대조 분석
- 서로 다른 관점의 노트 나란히 비교
- 합의점, 갈등점, 보완점 기록
- 표나 매트릭스로 시각화

### 3. 추상화
- 구체적 사례에서 일반 원칙 도출
- 상위 개념 노트 생성
- 하위 노트들과 링크

### 4. 질문 생성
- 기존 노트에서 답하지 못한 질문 도출
- 질문을 새로운 연구 방향으로 전환
- `#question` 태그로 관리

## NotebookLM 활용
- 관련 노트 그룹을 NotebookLM 소스로 동기화
- AI에게 종합 질문: "이 자료들의 공통 주장은?"
- AI 답변을 Obsidian에 새 노트로 기록
- 인간 검토 후 영구 노트로 승격

## 결과물
- 종합 노트 (`synthesis-[주제].md`)
- 새로운 연구 질문 목록
- 업데이트된 MOC
