# 문헌 리뷰 스킬

## 목적
특정 주제에 대한 기존 연구/지식을 체계적으로 조사하고 종합

## 절차

### 1단계: 검색
- 키워드 조합으로 자료 검색
- 스노우볼링: 핵심 논문의 참고문헌 추적
- 최소 10개 이상 자료 수집

### 2단계: 선별
- 관련성, 품질, 최신성 기준으로 필터링
- 각 자료에 [[source-note]] 생성

### 3단계: 분석
- 각 자료의 핵심 주장, 방법론, 결과 정리
- 공통점과 차이점 파악
- 연구 갭(gap) 식별

### 4단계: 종합
- 주제별 그룹화
- 시간순 흐름 정리
- 합의 영역 vs 논쟁 영역 구분

### 5단계: 기록
- MOC 노트에 리뷰 구조 정리
- 각 항목에서 영구 노트로 링크
- 결론 및 향후 연구 방향 기록
