# Obsidian 마크다운 문법 규칙

## 기본 문법
- 제목: `# H1` ~ `###### H6`
- 강조: `**굵게**`, `*기울임*`, `~~취소선~~`, `==하이라이트==`
- 목록: `-` 비순서, `1.` 순서, `- [ ]` 체크박스

## 내부 링크
- 위키링크: `[[노트이름]]`
- 별칭 링크: `[[노트이름|표시할 텍스트]]`
- 헤딩 링크: `[[노트이름#헤딩]]`
- 블록 링크: `[[노트이름#^블록id]]`

## 임베드
- 노트 임베드: `![[노트이름]]`
- 이미지 임베드: `![[이미지.png]]`
- PDF 임베드: `![[문서.pdf#page=5]]`

## 태그
- 인라인 태그: `#태그명`
- 중첩 태그: `#분류/하위분류`
- YAML 태그: frontmatter의 `tags:` 필드 사용

## Frontmatter (YAML)
```yaml
---
title: 노트 제목
tags: [태그1, 태그2]
date: YYYY-MM-DD
type: research | source | daily
status: draft | review | final
---
```

## 콜아웃
```markdown
> [!note] 제목
> 내용

> [!warning] 경고
> 경고 내용

> [!tip] 팁
> 팁 내용
```

## 코드 블록
- 인라인: `` `코드` ``
- 블록: ```` ```언어 ```` 로 감싸기

## 수학식
- 인라인: `$E = mc^2$`
- 블록: `$$` 로 감싸기
