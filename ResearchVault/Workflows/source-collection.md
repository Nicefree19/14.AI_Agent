# 자료 수집 절차

## 수집 소스
1. **학술 논문**: Google Scholar, arXiv, Semantic Scholar
2. **기술 문서**: 공식 문서, GitHub README
3. **블로그/글**: 기술 블로그, Medium, 개인 블로그
4. **영상**: YouTube, 컨퍼런스 발표
5. **서적**: 전자책, 물리책

## 수집 절차
1. 자료 발견 시 즉시 출처 노트 생성
2. [[source-note]] 템플릿 사용
3. URL, 저자, 날짜, 핵심 요약 기록
4. `#status/inbox` 태그 부여
5. `00-Inbox/` 폴더에 저장

## 품질 기준
- 출처 신뢰도 평가 (학술 > 공식문서 > 블로그)
- 발행일 확인 (최신성)
- 저자 전문성 확인
- 인용 횟수 참고 (논문의 경우)

## 정리 주기
- 매일: Inbox 노트 분류
- 매주: 미처리 노트 검토
- 매월: 불필요 노트 Archive로 이동
